# gogu

At first the code looks like a windows exe file, but when I tried running it with wine or with my windows vm it failes.
So I decided to doa xxd on it to make sure it is an exe. It wasn't. It was a ELF executable.

So I opened it in ida and looked in the main function. 
The main function was calling 3 functions from the same file (as far as I could tell).

By looking at them I saw that the last one (main_main_func3) as doing something with 69 chars (the lenght of the flag).
So I decided to put a breakpoint at `0x483e03` in gdb and read the values of `dil` since that one had the chars the function was decoding.

the chars I could get were in hex:
```python
0x63 0x74 0x66 0x7b 0x31 0x66 0x65 0x36 0x39 0x35 0x34 0x38 0x37 0x30 0x62 0x61 0x62 0x64 0x35 0x35 0x62 0x61 0x36 0x65 0x35 0x64 0x66 0x61 0x35 0x37 0x64 0x34 0x65 0x64 0x31 0x31 0x61 0x61 0x62 0x62 0x37 0x30 0x35 0x33 0x33 0x33 0x39 0x37 0x62 0x39 0x38 0x35 0x63 0x38 0x39 0x30 0x37 0x34 0x39 0x63 0x62 0x66 0x63 0x37 0x65 0x33 0x30 0x36 0x7d
```

Decoding the string from hex we get the flag.

flag: `ctf{1fe6954870babd55ba6e5dfa57d4ed11aabb70533397b985c890749cbfc7e306}`
